### Usage:

UI:

1. Per Day
   1. CRUD expenses at any day
   2. CRUD allocated budget(income) per day
2. Input (CRUD) expense category

Savings(Goal) = Income - Expenses

### Limitations

1.  Can only add expense for current day

### Entities

1. Expenses
2. Expense Type
3. Income - Income per month
4. Income Type:
   1. Values:
      1. Q - quarterly
      2. A
      3. M
      4. S
